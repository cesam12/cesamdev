# Sistema de Controle Financeiro

Projeto exemplo em Java com conexão MySQL (JDBC). Estrutura:

- src/: código fonte Java
- README.md: instruções

Como rodar (resumido):

1. Configurar banco MySQL e criar schema (ver scripts/sql/create_schema.sql)
2. Compilar: `javac -d bin src/**/*.java`
3. Rodar: `java -cp bin Main`
